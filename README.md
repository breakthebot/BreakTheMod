# **BreakTheMod**

[![Build Status](https://github.com/chariskar/breakthemod/actions/workflows/build.yml/badge.svg)](https://github.com/breakthebot/BreakTheMod/actions/workflows/build.yml)
![BreakTheMod Logo](/src/main/resources/assets/icon.png)

## **Description**

**BreakTheMod** enhances Minecraft gameplay by introducing a carefully selected set of commands inspired by BreakTheBot, along with brand-new features such as `whereIs` and `nearby`.

---

## **Features**

✅ **whereIs** – Locate a player based on the map API
✅ **nearby** – List players in visible (rendered) chunks, similar to what the map would show
✅ **locate** – Find the coordinates of a town or nation
✅ **goto** – Suggests which town to spawn in for closest proximity to another town
✅ **onlineStaff** – Lists all online staff members
✅ **townless** – Lists all online players without a town
✅ **coords** – Provides detailed coordinate information
✅ **discordLinked** – Checks whether a player’s EMC account is linked to their Discord account, including their Discord ID
✅ **nationpop** – Displays a nation’s population and nation bonus
✅ **Player Radar** – Shows nearby players using legal `/nearby` mechanics

## **Installation**

[**Download BreakTheMod**](https://modrinth.com/mod/breakthemod/)

### **Requirements**

* **Minecraft version:** `1.21.4`
* **Mod loader:** `Fabric`
* **Dependencies:**

  * [Fabric API](https://fabricmc.net/)
  * cloth-config2 >= 13.0.138
  * modmenu >= 9.2.0

### **Steps**

1. Download the mod from [Modrinth](https://modrinth.com/mod/breakthemod/).
2. Install Fabric (if you haven’t already).
3. Place the downloaded `.jar` file in your Minecraft `mods` folder.
4. Make sure the required dependencies are installed.
5. Launch Minecraft and enjoy!
